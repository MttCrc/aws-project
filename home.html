Relazione

Utilizzando il cloud di AWS creare un server virtuale che permetta di ospitare un sito web single page.

L'insieme degli applicativi che realizzeranno lo stack (server web, database, php) dovranno operare su docker.

Il sito web dovra' essere accessibile solo tramite connessione sicura https (con certificato self signed).

Il sito web accogliera' l'utente con un form di login che deve essere realizzato in maniera sicura (come abbiamo visto in classe) e solo se le credenziali sono corrette permettere la visione del sito web.

Il sito web dovra' essere scritto rispettando gli standard HTML5 e CSS3 e dovra' presentare la relazione di quanto fatto per la creazione del sito stesso: tecnologie utilizzate e loro spiegazione, scelte fatte, ecc.

Inoltre, dovra' essere "gradevole alla vista". Potete far uso di altre tecnologie se lo desiderate (JavaScript, bootstrap ecc).


Le tecnologie utilizzate sono state il cloud aws, docker, lamp, phpMyAdmin, php, html, css, javascript e sql


AWS

e' una piattaforma di cloud computing offerta da Amazon. Il cloud di AWS fornisce un'ampia gamma di servizi di infrastruttura informatica su richiesta, tra cui l'elaborazione, lo storage e il networking.


DOCKER

e' una piattaforma di virtualizzazione leggera e portabile che consente di creare e distribuire applicazioni in container software. In pratica, Docker consente di creare un ambiente isolato e indipendente per l'esecuzione di applicazioni, fornendo una sorta di "scatola" in cui l'applicazione e tutte le sue dipendenze (librerie, framework, etc.) possono essere eseguite.


LAMP

e' un acronimo che sta per Linux, Apache, MySQL e PHP, ed è un insieme di tecnologie utilizzate comunemente per creare e gestire server web. Insieme, queste tecnologie costituiscono un ambiente di sviluppo e hosting completo per la creazione e la gestione di applicazioni web dinamiche. LAMP è ampiamente utilizzato nella creazione di applicazioni web open source come WordPress, Joomla e Drupal.


PhpMyAdmin

e' un'applicazione web gratuita e open source che consente di gestire il database MySQL attraverso un'interfaccia utente grafica. Con PhpMyAdmin è possibile creare tabelle e campi, inserire, modificare ed eliminare dati, eseguire query, importare ed esportare file e molto altro.


PHP

e' un linguaggio di scripting lato server utilizzato principalmente per lo sviluppo di applicazioni web dinamiche e interattive.


HTML

e' un linguaggio di marcatura utilizzato per la creazione di pagine web. HTML definisce la struttura e i contenuti di una pagina web.


CSS

e' un linguaggio utilizzato per definire l'aspetto di una pagina web. CSS viene utilizzato per definire lo stile, il layout e l'aspetto generale di una pagina web.


JavaScript

e' un linguaggio di programmazione lato client utilizzato principalmente per lo sviluppo di applicazioni web interattive. JavaScript viene utilizzato per aggiungere funzionalità e interattività a una pagina web.


SQL

e' un linguaggio utilizzato per l'accesso e la manipolazione dei dati in un database relazionale. SQL viene utilizzato per creare, modificare e interrogare i dati in un database.
